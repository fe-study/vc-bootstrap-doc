## Contribution

clone this repo, then `pull request`, we will see and merge if necessary, thanks anyway~

## Developing

Start a dev server at `localhost:4000`:

```
$ npm install -g hexo-cli
$ npm install
$ hexo g
$ hexo deploy
$ hexo server
```
